To make the dashboard auto-update, publish the Summary tab once and paste its CSV link below.

  1. In Google Sheets: File → Share → Publish to web
  2. Under "Link", pick the Summary sheet and format Comma-separated values (.csv)
  3. Click Publish, copy the link, paste it here ↓

Also works with a gviz link: …/gviz/tq?tqx=out:csv&gid=… if the sheet is shared "anyone with the link can view".
